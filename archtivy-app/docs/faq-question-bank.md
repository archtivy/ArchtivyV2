# FAQ Question Bank

Questions real users ask (or would ask) when using Archtivy. Sourced from usability cases, support-style scenarios, and trust/credibility flows. Use to extend the live FAQ and reduce support load.

**Legend:** [V] Visitor / Reader · [D] Designer · [B] Brand · [R] Recruiter · [P] Power user · [A] Admin · [I] International · [M] Mobile
